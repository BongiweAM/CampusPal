# Campus Pal - TUT Student Assistant

A comprehensive mobile-friendly web application for TUT students, particularly first-year students, to manage their academic life.

## Features

### 1. Authentication System
- Secure login with student number and password
- Demo PIN system for demonstration purposes
- Forgot password functionality

### 2. Home Page
- Animated welcome message in all South African languages
- App functionality overview
- Modern interface with TUT branding

### 3. Timetable Management
- View daily/weekly class schedule
- One-click navigation to classrooms
- Class details (time, venue, lecturer)

### 4. Student Card Registration
- Online application for student card
- Upload required documents (ID, proof of registration)
- Photo upload/capture functionality
- Application tracking

### 5. Campus Map
- Interactive Soshanguve South Campus map
- Building search functionality
- Building details and directions
- Step-by-step navigation

### 6. Student Profile
- Personal information display
- Academic status
- Contact information
- Settings management

## Color Scheme
- **Primary**: Yellow (#FFD700)
- **Secondary**: TUT Blue (#0033A0) and Red (#EF3340)
- **Backgrounds**: Light gray (#F5F5F5)
